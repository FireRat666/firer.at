How to Embed Banter Games

Adding these games to your Banter space is simple. Just insert a <script> tag into your space's HTML pointing to the game's script URL.

Common Configuration Parameters

All games support the following URL query parameters to control their position, size, and layout inside virtual spaces:

Parameter Default Value Description
instance (Page URL) Recommended. A unique ID (e.g., ?instance=lobby-chess) to isolate game states if you have multiple games of the same type in your space.
boardPosition 0 1.1 0 The center position of the board in 3D world space coordinates (x y z).
boardRotation 0 0 0 The rotation angles of the board in Euler space (x y z).
boardScale 1 1 1 The overall scale of the 3D game objects.
hideUI false Set to true to hide the floating 3D control panels (reset/undo buttons).

Note: Coordinates are space-separated. In URLs, spaces must be encoded as %20 or +.
