Banter Games

A collection of multiplayer games designed for BanterVR. These games feature real-time synchronization, allowing you to play with friends in VR or on 2D screens.

Banter Checkers

A fully synchronized multiplayer checkers game. Features forced jumps, multi-jumps, and king promotion. Runs entirely client-side using Banter's space state.

Banter Chess

A serverless, 3D chess game. Handles core chess rules including move validation and FEN state management using chess.js. Supports multiple independent games in one space.

Banter Connect 4

A plug-and-play synchronized Connect 4 game. Drop pieces into columns and connect four to win. Features procedurally generated geometry and customizable board settings.

Banter Go (Weiqi)

A fully implemented Go game with standard rules including liberties, capturing, suicide prevention, and Ko rule. Includes a scoreboard and pass functionality.

Banter Gomoku

The classic Mancala (Kalah) game. Pick up stones and distribute them to capture your opponent's pieces. Features simple embedding and custom model support.
